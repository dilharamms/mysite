❓ COMMON QUESTIONS

Q: Where do I upload games?
A: In admin_playground_games.php -> "Add New Game" section

Q: What file types can I upload?
A: Games: HTML, PHP, ZIP | Images: JPG, PNG, GIF, WebP

Q: Can a game be in multiple categories?
A: Yes! Check multiple categories when adding a game

Q: Can I add more categories?
A: Yes! Go to admin_playground_categories.php and add custom categories

Q: How do I disable a game temporarily?
A: Click the power icon on the game card in the admin panel

Q: Are external games supported?
A: Yes! Choose "Link" as game type and enter the URL

Q: How do I delete a game?
A: Click the trash icon on the game card in the admin panel

Q: Can students upload games?
A: No, only admins can upload games via admin panel

Q: Is the playground responsive?
A: Yes! Works perfectly on desktop, tablet, and mobile

🐛 TROUBLESHOOTING

PROBLEM: Setup script doesn't work
SOLUTION: 
  1. Make sure PHP is in your PATH
  2. Check db_connect.php has correct database credentials
  3. Ensure MySQL is running

PROBLEM: Database tables not created
SOLUTION:
  1. Run the setup script again: php helpers/setup_playground_tables.php
  2. Check MySQL user has CREATE TABLE privileges
  3. Verify database name in db_connect.php

PROBLEM: Playground link not showing in navbar
SOLUTION:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Refresh the page (F5)
  3. Check navbar.php was properly modified

PROBLEM: Images not uploading
SOLUTION:
  1. Check folder permissions: chmod 755 assest/images/playground/
  2. Verify PHP upload_max_filesize setting in php.ini
  3. Check disk space is available

PROBLEM: Games not appearing in user view
SOLUTION:
  1. Make sure game status is "active" (green button in admin)
  2. Check category is marked as "active"
  3. Verify game is assigned to at least one category
  4. Clear browser cache and refresh

PROBLEM: Play button doesn't work
SOLUTION:
  1. For uploaded games: Check file path is correct
  2. For external links: Verify URL starts with http:// or https://
  3. Check game status is "active"

📊 DATABASE STRUCTURE

Three tables are created:

1. playground_categories
   - Stores category information
   - Includes grade levels and category icons
   - Can be marked active/inactive

2. playground_games
   - Stores game information
   - Tracks play count and ratings
   - Links to categories via junction table

3. playground_game_categories
   - Junction table for many-to-many relationship
   - Links games to multiple categories

Default data:
  - 14 categories inserted (8 grades + 6 general)
  - All marked as active
  - Ready for games to be added
